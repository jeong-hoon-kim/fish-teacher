# 🎣 스마트 어종 판독기 (Ocean Guard AI)

AI를 활용하여 낚은 물고기의 어종을 판독하고, 대한민국 수산자원관리법에 따른 금지체장 및 금어기 여부를 즉시 확인해주는 앱입니다.

## 🛠 실행 가이드

### 1. 백엔드 설정 (YOLOv8 모델 실행)
이 앱은 YOLOv8 모델(.pt)을 사용하여 어종을 분류합니다. 파이썬 환경이 필요합니다.

- **파일 위치**: `best.pt` 파일을 프로젝트 루트 폴더(`server.py`와 같은 위치)에 넣어주세요.
- **의존성 설치**:
  ```bash
  pip install -r requirements.txt
  ```
- **서버 실행**:
  ```bash
  python server.py
  ```
  *(서버는 http://localhost:8000 에서 실행됩니다)*

### 2. 프런트엔드 설정 (React 앱 실행)
사용자 UI와 측정 로직이 포함된 리액트 앱입니다.

- **의존성 설치**:
  ```bash
  npm install
  ```
- **앱 실행**:
  ```bash
  npm run dev
  ```
  *(브라우저에서 http://localhost:3000 으로 접속하세요)*

## 📏 주요 기능
- **AI 어종 판독**: `best.pt` 모델을 통한 정확한 실물 판독
- **정밀 길이 측정**: 신용카드(8.56cm)를 기준점으로 사용하여 사진 속 물고기 길이를 cm 단위로 계산
- **법규 자동 확인**: `regulation.json`의 최신 금지체장/금어기 데이터를 바탕으로 방생 대상 여부 판정

## ⚠️ 주의사항
- 사진 촬영 시 물고기와 카드가 수평이 되도록 촬영해야 가장 정확합니다.
- 본 앱의 결과는 참고용이며, 실제 법적 책임은 낚시인 본인에게 있습니다.
